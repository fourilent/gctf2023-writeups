## Pre-requisites for scripts

- Linux (tested on Kali Linux 2023.3)
- Python >3.9
- pip
- virtualenv
- p7zip(-full)
- fdupes
- steghide

Then run

```shell
python -m venv .venv && \
source .venv/bin/activate && \
pip install -r requirements.txt
```
